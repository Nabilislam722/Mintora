# init



Create a `.env` file on the root directory with following values

``
R2_ACCESS_KEY_ID=
R2_SECRET_ACCESS_KEY=
R2_PUBLIC_URL=

``

TO RUN Must Start the backend server first

``
node ./server/crawler.js
node ./server/index.js
node ./server/sync.js AddressOfTHeNFT  #When listing new nfts

``
