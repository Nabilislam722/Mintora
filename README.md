# Create a `.env` file on the root directory with following values

```
R2_ACCESS_KEY_ID=
R2_SECRET_ACCESS_KEY=
R2_PUBLIC_URL=

```
<sub>MUST HAVE CLOUDFLARE R2</sub>

# TO RUN, Must Do the steps above ⚒️
```
yarn install
```

```
yarn crawler
```
```

```
